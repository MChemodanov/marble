MarbleDataFacade Class Reference

from PyKDE4.marble import *

Namespace: Marble.MarbleDataFacade

Detailed Description

Abstract class:
This class can be used as a base class for new classes, but can not be instantiated directly.

A read-only interface to internal Marble data.

This class provides a read-only interface to some internal data in Marble. Its intended use is for plugins that should be able to access the data in question, but not change it.
